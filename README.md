# Obsidian Workflow Hub

This vault is designed to work with Claude Code as the primary interface.

## Folder Structure

```
_inbox/          → Quick capture, unsorted notes
daily/           → Daily briefs and working notes
weekly/          → Weekly reviews and planning
projects/        → Active project workspaces
  ├── salesforce/
  ├── tray/
  └── [other]/
meetings/        → Meeting summaries and action items
reference/       → SOPs, documentation, templates
tasks/           → Task deep-dives and reviews
archive/         → Completed projects
```

## Common Commands (Claude Code)

### Morning Startup
```
"What's my day look like?"
"Generate my daily brief"
```

### Task Management
```
"What tasks are due this week?"
"Show me details on [task name]"
"Mark [task] complete with note: [note]"
```

### Project Work
```
"Let's work on [project name]"
"Create a new project for [description]"
```

### Meeting Follow-up
```
"What action items came from my recent meetings?"
"Create tasks from today's meeting notes"
```

## File Naming Conventions

- Daily notes: `YYYY-MM-DD.md`
- Weekly reviews: `YYYY-WXX.md`
- Meeting notes: `YYYY-MM-DD-[meeting-name].md`
- Projects: lowercase-with-dashes/
