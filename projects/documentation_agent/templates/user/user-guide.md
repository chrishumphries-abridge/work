## Configuration

[Product Name] can be configured via:

### Configuration File

Create a `config.json` (or `.product-namerc`) in your project root:

```json
{
  "setting1": "value",
  "setting2": true,
  "setting3": {
    "nested": "value"
  }
}
```

### Environment Variables

| Variable | Description |
|----------|-------------|
| `PRODUCT_SETTING1` | Overrides setting1 |
| `PRODUCT_DEBUG` | Enable debug mode |

### Command-line Flags

Most settings can also be passed as flags:

```bash
product-name command --setting1 value
```
